IntelliSS7 Function MTP3_Message::SetPayload


NAME 
    MTP3_Message::SetPayload()
SCOPE 
    public
SYNOPSIS 
    int
    MTP3_Message::SetPayload(const ITS_OCTET* const data, const int len);
PURPOSE 
   Set the data portion of an MTP3 message
INPUT PARAMETERS 
    data - the octet vector to copy in
    len - the size of the vector
INPUT/OUTPUT PARAMETERS 
    None.
OUTPUT PARAMETERS 
    None.
RETURN VALUE 
    ITS_SUCCESS - value successfully set.  Any other return
        code indicates an IntelliSS7 processing error.
