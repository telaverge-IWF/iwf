IntelliSS7 Function TCAP_Invoke::GetLinkedID


NAME 
    TCAP_Invoke::GetLinkedID()
SCOPE 
    public
SYNOPSIS 
    ITS_OCTET
    TCAP_Invoke::GetLinkedID() const;
PURPOSE 
    This method gets the linked-id (ITU) or correlation-id (ANSI) of an
    invoke, if supplied.
RETURN VALUE 
    The linked id as an unsigned char.
NOTES 
    If the invoke does not have a linked id, an exception will be
    thrown.
