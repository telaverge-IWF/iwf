INTERFACE 
        TCAP_Abort
PURPOSE 
        The TCAP_Abort message class defines the basic interface for aborting
        TCAP transactions.  Note that this class can be user generated for
        ITU TCAP, but is stack-generated only with ANSI TCAP.
