IntelliSS7 Function SCCP_Cause::SCCP_Cause


NAME 
    SCCP_Cause::SCCP_Cause()
SCOPE 
    public
SYNOPSIS 
    SCCP_Cause::SCCP_Cause(const ITS_OCTET id, const ITS_OCTET cause);
PURPOSE 
    This method constructs a cause object.
INPUT PARAMETERS 
    id - one of SCCP_PRM_RETURN_CAUSE, SCCP_PRM_REFUSAL_CAUSE,
        SCCP_PRM_ERROR_CAUSE, SCCP_PRM_RELEASE_CAUSE, or
        SCCP_PRM_RESET_CAUSE.
    cause - the cause value.
INPUT/OUTPUT PARAMETERS 
    None.
OUTPUT PARAMETERS 
    None.
RETURN VALUE 
    None.
