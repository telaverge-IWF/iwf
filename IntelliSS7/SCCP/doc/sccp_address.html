INTERFACE 
        SCCP_Address
PURPOSE 
        SCCP address intermediate class.
USAGE 
        This class is an intermediate class and should not be used directly.
        The SCCP_CallingPartyAddr and SCCP_CalledPartyAddre IE(s) are derived
        from this class.
