IntelliSS7 Function SCCP_Address::IsRoutedByPCSSN


NAME 
    SCCP_Address::IsRoutedByPCSSN()
SCOPE 
    public
SYNOPSIS 
    bool
    SCCP_Address::IsRoutedByPCSSN() const;
PURPOSE 
    This method determines if the "routed by" flag value for an
    SCCP_Address object has been set.
INPUT PARAMETERS 
    None.
INPUT/OUTPUT PARAMETERS 
    None.
OUTPUT PARAMETERS 
    None.
RETURN VALUE 
    true if the value has been set.  false otherwise.
NOTES 
    Setting this to false implies routing by GTT.
