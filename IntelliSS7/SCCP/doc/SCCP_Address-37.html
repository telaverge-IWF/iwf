IntelliSS7 Function SCCP_Address::SetRouteByPCSSN


NAME 
    SCCP_Address::SetRouteByPCSSN()
SCOPE 
    public
SYNOPSIS 
    void
    SCCP_Address::SetRouteByPCSSN(const bool which);
PURPOSE 
    This method sets the "routed by" flag value for an SCCP_Address
    object.
INPUT PARAMETERS 
    which - the value to store for the "routed by" flag
INPUT/OUTPUT PARAMETERS 
    None.
OUTPUT PARAMETERS 
    None.
RETURN VALUE 
    None.
NOTES 
    Setting this to false implies routing by GTT.
