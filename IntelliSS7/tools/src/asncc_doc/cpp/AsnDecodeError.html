Class AsnDecodeError

The class AsnDecodeError is used to indicate an ASN.1 error during the decoding phase of an ASN.1 encoded value (i.e. call to the Decode member function). An ASN.1 decode error indicates a misuse of an ASN.1 type in the remote application code (e.g. invalid octet string size, missing mandatory element in a set or a sequence) or a corrupted ASN.1 encoded value. An instance of (i.e. object) the class AsnDecodeError is thrown (C++ exception) each time an ASN.1 decode error is detected. The AsnDecodeError object contains a full description and the precise location of the error. The class AsnDecodeError inherits from the class ITS_GenericException (part of IntelliSS7 Support classes). See IntelliSS7 Support classes documentation for more information. The detection of some ASN.1 decode errors can be turned off at runtime using the class AsnOptions.

AsnError GetError() The GetError member function returns one of the error constants of the AsnError C++ enumerated type.
