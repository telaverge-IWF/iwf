Class AsnObject

The class AsnObject is the abstract base class of all classes representing an ASN.1 built-in type. It is not intended to be used directly by the end-user.

The class AsnObject is the base class of the following classes that represent the ASN.1 built-in types:

AsnAny (built-in type ANY)

AsnBitString (built-in type BIT STRING)

AsnBoolean (built-in type BOOLEAN)

AsnChoice (built-in type CHOICE)

AsnEnumerated (built-in type ENUMERATED)

AsnInteger (built-in type INTEGER)

AsnNull (built-in type NULL)

AsnObjectIdentifier (built-in type OBJECT IDENTIFIER)

AsnOctetString (built-in type OCTET STRING)

AsnReal (built-in type REAL)

AsnSequence (built-in type SEQUENCE)

AsnSequenceOf (built-in type SEQUENCE OF)

AsnSet (built-in type SET)

AsnSetOf (built-in type SET OF)
