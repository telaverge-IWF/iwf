Class AsnAny

The class AsnAny is the abstraction of the ASN.1 built-in type ANY. All classes representing an ASN.1 type based on the ASN.1 built-in type ANY inherit from it. The class AsnAny itself inherits from the abstract base class AsnObject. It is not intended to be used directly by the end-user (only the heirs of the class AsnAny should be directly used).
