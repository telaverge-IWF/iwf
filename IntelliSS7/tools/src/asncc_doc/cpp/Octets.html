Class Octets

The class Octets allows handling encoded ASN.1 data values (vector of bytes). It is directly based (i.e. containment relationship) on the class std::vector<byte>, but provides additional services required by the ASN.1 runtime (e.g. cursor used by the various encode/decode algorithms). These additional services are not intended to be used by the end-user. The end-user should only use the service of the public interface described on this page.
