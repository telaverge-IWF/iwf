Class ITS_Postcondition

The class ITS_Postcondition is related to IntelliSS7 assertion mechanism. If during execution time an assertion fails (precondition, postcondition or assertion) a corresponding C++ exception is thrown. The class ITS_Postcondition inherits from the class ITS_GenericException (part of IntelliSS7 Support classes). A failed postcondition most likely denotes an error inside the ASN.1 runtime itself. See IntelliSS7 Support classes documentation for more information on ITS_GenericException and ITS_Postcondition.
