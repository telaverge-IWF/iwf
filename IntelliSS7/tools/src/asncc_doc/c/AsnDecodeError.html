Class Interface AsnDecodeError

The class interface AsnDecodeError is used to indicate an ASN.1 error during the decoding phase of an ASN.1 encoded value (i.e. call to the associated __Decode function). An ASN.1 decode error indicates a misuse of an ASN.1 type in the remote application code (e.g. invalid octet string size, missing mandatory element in a set or a sequence) or a corrupted ASN.1 encoded value. An instance of (i.e. object) the class interface AsnDecodeError is returned each time an ASN.1 decode error is detected (returned in association with the return error code ITS_E_ASN_DECODE_ERROR). The AsnDecodeError object contains a full description and the precise location of the error. The detection of some ASN.1 decode errors can be turned off at runtime using the relevant ASN_Options__ functions.
