Class Interface ASN_Object

The class interface ASN_Object is the abstract base class of all class interfaces representing an ASN.1 built-in type. It is not intended to be used directly by the end-user.

The class interface ASN_Object is the base class interface of the following class interfaces that represent the ASN.1 built-in types:

ASN_Any (built-in type ANY)

ASN_BitString (built-in type BIT STRING)

ASN_Boolean (built-in type BOOLEAN)

ASN_Choice (built-in type CHOICE)

ASN_Enumerated (built-in type ENUMERATED)

ASN_Integer (built-in type INTEGER)

ASN_Null (built-in type NULL)

ASN_ObjectIdentifier (built-in type OBJECT IDENTIFIER)

ASN_OctetString (built-in type OCTET STRING)

ASN_Real (built-in type REAL)

ASN_Sequence (built-in type SEQUENCE)

ASN_SequenceOf (built-in type SEQUENCE OF)

ASN_Set (built-in type SET)

ASN_SetOf (built-in type SET OF)
