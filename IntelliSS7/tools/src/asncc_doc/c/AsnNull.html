Class Interface ASN_Null

The class interface ASN_Null is the abstraction of the ASN.1 built-in type NULL. All class interfaces representing an ASN.1 type based on the ASN.1 built-in type NULL inherit (conceptually) from it. The class interface ASN_Null itself inherits (conceptually) from the abstract base class interface ASN_Object. It is not intended to be used directly by the end-user (only the heirs of the class interface ASN_Null should be directly used).
