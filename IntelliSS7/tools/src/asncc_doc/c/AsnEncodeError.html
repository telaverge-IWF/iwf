Class Interface AsnEncodeError

The class interface AsnEncodeError is used to indicate an ASN.1 error during the encoding phase of an ASN.1 value (i.e. call to the associated __Encode function). An ASN.1 encode error always indicates a misuse of an ASN.1 type in the application code written by the end-user (e.g. invalid octet string size, missing mandatory element in a set or a sequence). An instance of (i.e. object) the class interface AsnEncodeError is returned each time an ASN.1 encode error is detected (returned in association with the return error code ITS_E_ASN_ENCODE_ERROR). The AsnEncodeError object contains a full description and the precise location of the error. The detection of some ASN.1 encode errors can be turned off at runtime using the relevant ASN_Options__ functions.
