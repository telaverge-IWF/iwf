Class Interface ASN_Octets

The class interface ASN_Octets allows handling encoded ASN.1 data values (array of bytes). The public interface is divided in two parts. The first part provides services allowing to store and retrieve encoded ASN.1 data values. The second part of the public interface provides additional services required by the ASN.1 runtime. These additional services are not intended to be used by the end-user. The end-user should only use the services of the first part of the public interface (described in this page).
