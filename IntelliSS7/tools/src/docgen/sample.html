IntelliSS7 Interface

INTERFACE 
        ITS_Event
PURPOSE 
        The ITS_Event is the heart of the C++ interface to IntelliSS7.  All
        intertask and interprocess indications are passed between entities
        in the ITS_Event structure.  The class wrappers the C structure
        ITS_EVENT, which defines the basic structure of an indication.
USAGE 
        User code should use objects of this class when receiving events
        (normally via ITS_Worker::GetEvent() or ITS_Transport::GetEvent()).
        Users should not (in general) create ITS_Events for transmitting
        information; all IntelliSS7 APIs provide routines for building
        and sending ITS_Events, as the internal structure of the ITS_EVENT
        contents are not documented.

IntelliSS7 Function: ITS_Event::ITS_Event()




NAME

    ITS_Event::ITS_Event()


SCOPE

    public


SYNOPSIS

    ITS_Event::ITS_Event(ITS_USHORT src, ITS_USHORT len, ITS_OCTET *data);



PURPOSE

     This constructor for ITS_Events is used by most internal
     IntelliSS7 code to build an ITS_Event for transmission.
     While it can be invoked by user code, it's use is not
     recommended.



INPUT PARAMETERS

     src - a transport identifier.  Indicates the *originating*
         transport of an event.
     len - the length of the data portion of this event.
     data - the data payload to be carried by this event.



INPUT/OUTPUT PARAMETERS

     None.



OUTPUT PARAMETERS

     None.



RETURN VALUE

     None.

IntelliSS7 Function: ITS_Event::InitializeEventPool()




NAME

    ITS_Event::InitializeEventPool()


SCOPE

    public


SYNOPSIS

    int
    ITS_Event::InitializeEventPool();



PURPOSE

    This method creates a fixed-size, process-wide pool of ITS_Event's.
    The pool is then used for all intertask communications (that is,
    the pool represents the available slots for messages when
    communicating between tasks in a process).



INPUT PARAMETERS

    None.



INPUT/OUTPUT PARAMETERS

    None.



OUTPUT PARAMETERS

    None.



RETURN VALUE

    Returns ITS_SUCCESS if the pool was successfully initialized.
    Any other return value indicates an IntelliSS7 error code.



NOTES

IntelliSS7 Function: ITS_Event::Clone()




NAME

    ITS_Event::Clone()


SCOPE

    public


SYNOPSIS

    ITS_ListEntry *
    ITS_Event::Clone() const;



PURPOSE

    This method allocates an ITS_Event from the process-wide event
    pool and copies the contents of the calling object into the
    newly allocated event, thus creating a "clone" of the calling
    object.



INPUT PARAMETERS

    None.



INPUT/OUTPUT PARAMETERS

    None.



OUTPUT PARAMETERS

    None.



RETURN VALUE

    A pointer to the cloned entry, or NULL if the entry could not
    be cloned.



NOTES

    This method should be used ONLY by task queue manipulation
    code; allocating an event via this method from user code removes
    that entry from the pool available for intertask messages.
